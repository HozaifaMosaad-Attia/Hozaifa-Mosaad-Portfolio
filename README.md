## 🛠️ Tech Stack
This portfolio was built using modern web technologies:
* **HTML5 & CSS3**: Semantic structure and advanced styling.
* **JavaScript (ES6+)**: Interactive elements and scroll logic.
* **AOS Library**: Smooth "Animate On Scroll" effects.
* **Responsive Design**: Fully optimized for Mobile, Tablet, and Desktop using CSS Clamp.

## ✨ Key Features
* **Dynamic Navigation**: Active link highlighting based on scroll position.
* **Glassmorphism**: Modern UI using backdrop filters.
* **Interactive Menu**: Smooth mobile hamburger menu with CSS transitions.
* **Contact Form**: Integrated and ready for user inquiries.

- `js/main.js`: Scroll logic and AOS initialization.
